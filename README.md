# Finanzas con Manuel — Sitio web

Sitio estático responsive construido a partir del Manual Maestro de Diseño.

## Archivos
- `index.html`: Home completa.
- `styles.css`: sistema visual y responsive.
- `script.js`: menú móvil, animaciones y microinteracciones.
- `blog.html`: estructura inicial del centro de recursos.
- `privacy.html`: plantilla de aviso de privacidad.
- `assets/logo.svg`: logotipo editable.
- `assets/hero-placeholder.svg`: marcador visual para sustituir por la fotografía profesional.

## Publicación
Puede publicarse en Netlify, Vercel, GitHub Pages o integrarse a Wix/WordPress como referencia.

## Ajustes obligatorios antes de publicar
1. Sustituir `assets/hero-placeholder.svg` por una fotografía profesional de Manuel.
2. Revisar el aviso de privacidad con un especialista legal.
3. Completar los artículos del blog.
4. Confirmar enlaces de WhatsApp, Instagram y Calendly.
5. Conectar dominio, Analytics y Search Console.
6. Comprimir imágenes en WebP.
